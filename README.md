# Sales Data Fetcher (MERN)

A simple MERN backend project that fetches sales data from the Petpooja API and stores it in MongoDB.

## Features

- Fetch sales data from Petpooja API
- Save records into MongoDB collection
- Simple GET API endpoint to trigger data fetch
